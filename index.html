-- Итоговое приложение на чистом js:
- Заходим на страницу
- кнопка получить данные
- по клику идем за данными - на странице появляется список - кнопка "вызвать окно alert"
- кнопка "убрать обработчик с кнопки вызвать окно alert"
- строка поиска - при вводе в нее фильтруется полученный список
- все данные записываются в localStorage, то есть после закрытия браузера - у меня сохранится результат поиска
